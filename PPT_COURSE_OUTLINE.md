### 视频内容包括：
1. **传统方法 vs 列表推导式**：

   传统方法：
   ```python
   squares = []
   for i in range(1, 6):
       squares.append(i**2)
   ```

   列表推导式：
   ```python
   squares = [i**2 for i in range(1, 6)]
   ```

   结果都是：[1, 4, 9, 16, 25]

2. **带条件的列表推导式**：
   ```python
   evens = [i for i in range(1, 11) if i % 2 == 0]
   # 结果: [2, 4, 6, 8, 10]
   ```
   可视化：显示过滤过程

### 讲解要点：
- 列表推导式更简洁、高效
- 基本语法：`[expression for item in iterable if condition]`
- 不要过度复杂化，保持可读性
- 时间：4分钟

## PPT第17页：引用、浅复制、深复制
**展示方式：视频动画**

### 视频文件：
运行 `list_copy_animation.py` 生成完整视频

### 视频内容包括：
1. **引用赋值（=）**：
   ```python
   list1 = [1, 2, 3]
   list2 = list1  # 指向同一对象
   list2[0] = 99
   # list1 和 list2 都变成 [99, 2, 3]
   ```
   可视化：两个变量指向同一个列表对象

2. **浅复制（.copy() 或 [:]）**：
   ```python
   list1 = [1, 2, 3]
   list2 = list1.copy()  # 创建新对象
   list2[0] = 99
   # list1 仍是 [1, 2, 3]
   # list2 是 [99, 2, 3]
   ```
   可视化：两个独立的列表对象

3. **深复制（copy.deepcopy()）**：
   ```python
   import copy
   list1 = [[1, 2], [3, 4]]
   list2 = copy.deepcopy(list1)
   # 完全独立，包括嵌套对象
   ```

### 讲解要点：
- 这是Python中最常见的坑之一
- 引用赋值不创建新对象
- 浅复制只复制外层，嵌套对象仍共享
- 深复制递归复制所有层级
- 时间：3分钟

## PPT第18页：常见陷阱和最佳实践
**展示方式：纯文字PPT + 代码示例**

### 内容：

#### 常见陷阱：

1. **修改循环中的列表**：
   ```python
   # 错误示例
   numbers = [1, 2, 3, 4, 5]
   for num in numbers:
       if num % 2 == 0:
           numbers.remove(num)  # 危险！可能跳过元素

   # 正确方法
   numbers = [num for num in numbers if num % 2 != 0]
   ```

2. **默认参数陷阱**：
   ```python
   # 错误示例
   def add_item(item, lst=[]):
       lst.append(item)
       return lst

   # 正确方法
   def add_item(item, lst=None):
       if lst is None:
           lst = []
       lst.append(item)
       return lst
   ```

3. **列表乘法的坑**：
   ```python
   # 错误示例
   matrix = [[0] * 3] * 3  # 所有行都是同一对象！

   # 正确方法
   matrix = [[0] * 3 for _ in range(3)]
   ```

#### 最佳实践：

- 优先使用列表推导式，代码更简洁
- 需要修改列表时避免在循环中操作
- 记住列表是可变对象，传参时注意副作用
- 大数据量时考虑使用生成器而非列表
- 使用type hints提高代码可读性：`def process(items: list[int]) -> list[int]:`

### 讲解要点：
- 这些是实际项目中最容易犯的错误
- 分享自己的踩坑经验
- 时间：1分钟
